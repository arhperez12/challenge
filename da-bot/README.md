# Бот для интеграции DonationAlerts с EGE Challenge

Этот микросервис (скрипт) подключается к серверам DonationAlerts по протоколу Centrifugo (WebSockets) и слушает донаты в реальном времени. Когда приходит определенный донат, бот отправляет скрытый API-запрос к вашей запущенной игре Next.js, чтобы активировать нужный эффект.

## Настройка

Вам потребуется создать файл `.env` рядом с этим скриптом и указать в нем токен от DonationAlerts.

### Где взять токен?
1. Зайдите в ваш аккаунт DonationAlerts.
2. Перейдите в раздел **"Настройки" -> "API"**.
3. Создайте новое приложение.
4. В качестве **Секретного токена** для Centrifugo берите значение, которое там указано (или используйте токен из URL виджета оповещений, если API сложно настроить). Проще всего взять токен из виджета: `https://www.donationalerts.com/widget/alerts?group_id=1&token=ВАШ_ТОКЕН_ЗДЕСЬ`

## Как запустить на сервере:

1. Скопируйте папку с ботом на сервер.
2. Установите зависимости: `npm install`
3. Запустите бота через PM2:
   ```bash
   pm2 start index.js --name da-bot
   ```

## Как работают команды:

В тексте доната зритель должен написать одну из команд:
- `!тряска` - Трясет экран всем
- `!блюр` - Мылит экран учителю (или `!блюр ученики`)
- `!переворот` - Переворачивает экран
- `!хинт` - Выводит сообщение из доната как подсказку
- `!минус_время` - Отнимает 30 секунд от таймера
- `!плюс_время` - Добавляет 30 секунд

*(Цены за каждый донат и условия можно будет легко настроить в коде файла `index.js`)*
