## Как обновить игру (загрузить новые изменения)

Если мы добавили новые функции в код, вам нужно стянуть эти изменения на сервер и перезапустить игру. Выполните следующие команды по очереди после входа на сервер:

1. **Перейдите в папку с проектом:**
   ```bash
   cd /root/challenge
   ```

2. **Скачайте последние изменения из GitHub:**
   ```bash
   git pull
   ```

3. **Скомпилируйте обновленный проект:**
   ```bash
   npm run build
   ```
   *(Этот процесс занимает около минуты)*

4. **Перезапустите приложение, чтобы изменения вступили в силу:**
   ```bash
   pm2 restart ege-game
   ```

## Другие полезные команды (PM2)

Проект работает на сервере в фоновом режиме с помощью диспетчера процессов **PM2**.

*   **Посмотреть статус игры (работает ли она):**
    ```bash
    pm2 status ege-game
    ```

*   **Остановить игру:**
    ```bash
    pm2 stop ege-game
    ```

*   **Запустить игру (если она была остановлена):**
    ```bash
    pm2 start ege-game
    ```

*   **Посмотреть логи (ошибки консоли в реальном времени):**
    ```bash
    pm2 logs ege-game
    ```
    *(Чтобы выйти из логов, нажмите `Ctrl + C`)*
